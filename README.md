## 🇬🇧 English

A modern, high-performance dental clinic landing page built with **pure HTML, CSS, and Vanilla JavaScript** — no frameworks, no dependencies.

### ✨ Features

- ⚡ **Zero-dependency** — Pure HTML5, CSS3, Vanilla JS
- 📱 **Fully Responsive** — 480px / 768px / 1024px / 1280px+ breakpoints
- 🎨 **Premium UI** — Glassmorphism, micro-animations, navy + gold palette
- 🔍 **SEO Ready** — Schema.org (Dentist), Open Graph, Twitter Card, Local SEO
- 🗂️ **Modular** — Separated `index.html`, `style.css`, `app.js`
- 🏎️ **Performance** — Deferred JS, font preconnect, browser caching ready
- 💬 **WhatsApp CTA** — Appointment form with pre-filled WhatsApp message

### 📁 Project Structure

```
/
├── index.html       # Semantic, SEO-ready markup
├── style.css        # 4000+ lines, fully organized
├── app.js           # Scroll, menu, animations, counters
└── assets/
    └── logo-*.png
```

### 🚀 Getting Started

```bash
git clone https://github.com/yahyaaykan/Dis-Klinik-Web.git
cd Dis-Klinik-Web
python -m http.server 8000
# Open → http://localhost:8000
```

### 📱 Responsive Breakpoints

| Breakpoint | Target Device |
|-----------|--------------|
| `> 1024px` | Desktop |
| `≤ 1024px` | Tablet |
| `≤ 768px`  | Mobile |
| `≤ 480px`  | Small Phone (iPhone SE etc.) |

## 🇹🇷 Türkçe

Saf **HTML, CSS ve Vanilla JavaScript** ile geliştirilmiş modern ve yüksek performanslı bir diş kliniği açılış sayfası — framework yok, bağımlılık yok.

### ✨ Özellikler

- ⚡ **Sıfır bağımlılık** — Yalnızca HTML5, CSS3 ve Vanilla JS
- 📱 **Tam Responsive** — 480px / 768px / 1024px / 1280px+ kırılım noktaları
- 🎨 **Premium Tasarım** — Glassmorphism, mikro animasyonlar, lacivert + altın renk paleti
- 🔍 **SEO Optimizasyonu** — Schema.org (Dentist), Open Graph, Twitter Card, Yerel SEO
- 🗂️ **Modüler Yapı** — Ayrık `index.html`, `style.css`, `app.js` dosyaları
- 🏎️ **Performans** — Ertelenmiş JS yüklemesi, font preconnect, tarayıcı önbelleği uyumlu
