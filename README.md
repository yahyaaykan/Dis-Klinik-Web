# 🦷 Diş Klinik Web — Premium Dental Clinic Website

A modern, high-performance dental clinic landing page built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

> **Live Demo:** [View Project](https://yahyaaykan.github.io/Dis-Klinik-Web) *(deploy edilince aktif olur)*

---

## ✨ Features

- ⚡ **Zero-dependency** — Pure HTML5, CSS3, Vanilla JS
- 📱 **Fully Responsive** — Mobile (375px), Tablet (768px), Desktop (1280px+)
- 🎨 **Premium UI** — Glassmorphism, smooth animations, dark navy + gold palette
- 🔍 **SEO Optimized** — Schema.org (Dentist), Open Graph, Twitter Card, Local SEO meta tags
- 🗂️ **Modular Structure** — Separated `index.html`, `style.css`, `app.js`
- 🏎️ **Performance** — Deferred JS, font preconnect, browser caching ready
- 🗺️ **Google Maps** embedded iframe
- 💬 **WhatsApp CTA** — Direct WhatsApp message with pre-filled appointment form

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/yahyaaykan/Dis-Klinik-Web.git

# Run locally (Python)
cd Dis-Klinik-Web
python -m http.server 8000

# Open in browser
# http://localhost:8000
```

## 🛠️ Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript (ES6+)** — IntersectionObserver, scroll events, DOM manipulation
- **Google Fonts** — Cormorant Garamond + Outfit
- **Schema.org** — Dentist structured data
